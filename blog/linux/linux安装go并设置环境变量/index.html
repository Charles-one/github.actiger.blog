Linux安装go并设置环境变量

March 13, 2019

Mac用户直接使用brew install go即可

Linux用户

wget https://dl.google.com/go/go1.12.linux-amd64.tar.gz

tar -C /usr/local -xzf go1.12.linux-amd64.tar.gz

root用户去编辑vim /etc/profile中添加如下信息

export PATH=$PATH:/usr/local/go/bin

用户使用go需要 重新生效这个文件

source /etc/profile
