使用docker搭建nextcloud

May 17, 2019

1. nextcloud 个人私有云, 可以像网盘一样同步个人文件, 远程访问下载等.

2. 使用docker-compose搭建

使用 docker-compose-repository搭建


git clone https://github.com/Charles-one/docker-compose-repository.git ~/docker-compose-repository

cp -r ~/docker-compose-repository/nextcloud ~/nextcloud

cd ~/nextcloud

# 自动下载并启动容器

dc up -d

访问地址

http://服务ip地址
